# Lab4_Scripting_en_Sistemas_Operativos

Este repositorio contiene la resolución del **Laboratorio 4 de Sistemas Operativos**, enfocado en el uso de scripting para la automatización y administración básica del sistema operativo utilizando **PowerShell**.

## Contenido del repositorio

- **Reto1.ps1**  
  Script que muestra información básica del sistema: uso de CPU, memoria RAM y espacio en disco.

- **Reto2.ps1**  
  Script para la gestión de procesos, permitiendo listar procesos activos, identificar su PID y finalizar un proceso con confirmación previa.

- **Reto3.ps1**  
  Script de automatización que crea un directorio utilizando la fecha actual como parte del nombre.

- **Reto4.ps1**  
  Script interactivo que integra los retos anteriores mediante un menú, permitiendo al usuario seleccionar distintas opciones.

## Requisitos

- Sistema operativo Windows
- PowerShell
- Permisos de ejecución habilitados para scripts (`Set-ExecutionPolicy`)

## Ejecución

Los scripts pueden ejecutarse desde la terminal de PowerShell o desde Visual Studio Code:

```powershell
.\Reto1.ps1
.\Reto2.ps1
.\Reto3.ps1
.\Reto4.ps1
